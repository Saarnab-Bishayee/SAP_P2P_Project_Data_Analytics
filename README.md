# SAP Procure-to-Pay (P2P) Project

## Overview
This project demonstrates the Procure-to-Pay (P2P) cycle using SAP Materials Management (MM) in a simulation-based environment. It explains the complete procurement lifecycle from vendor creation to payment processing.

## Organization
The project is based on a fictitious company, **XYZ Manufacturing Pvt. Ltd.**, which deals in electronic components and requires an efficient procurement system.

## Process Flow
The P2P cycle includes:

- Vendor Creation  
- Purchase Requisition (PR)  
- Purchase Order (PO)  
- Goods Receipt (GR)  
- Invoice Verification  
- Payment Processing  

## Features
- End-to-end P2P workflow explanation  
- SAP MM and FI module integration  
- Business-oriented process representation
